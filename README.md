# file_server 
A simple file server written in rust
## Usage
```
$ file_server.exe --help
Easy http file server writed by rust.

Usage: file_server.exe [OPTIONS]

Options:
      --host <host>      [default: 0.0.0.0]
      --port <port>      [default: 8080]
      --path <path>      [default: ./]
  -c, --config <config>
  -h, --help             Print help
  -V, --version          Print version
```
